# 🍛 Sistema de Gestión de Comedor Estudiantil

## Descripción del Proyecto
[cite_start]Es una solución web desarrollada para automatizar el registro de almuerzos en una institución educativa[cite: 1, 27]. [cite_start]El sistema permite la entrada de datos de forma rápida y ofrece un contador dinámico para llevar el control diario[cite: 1, 28].

## Características principales
* [cite_start]**Interfaz Intuitiva:** Formulario diseñado para que el usuario ingrese nombre y cédula en segundos[cite: 1, 30].
* [cite_start]**Conexión en Tiempo Real:** Integración con base de datos para el almacenamiento seguro de la información[cite: 1, 31].
* [cite_start]**Módulo Administrativo:** Acceso restringido para la visualización de reportes y listados[cite: 1, 32].

## Tecnologías utilizadas
* [cite_start]**HTML5 / CSS3:** Estructura y diseño visual[cite: 1, 34].
* [cite_start]**JavaScript (ES6):** Lógica del lado del cliente y manejo de eventos[cite: 1, 35].
